Dokumen Pengesahan Proyek Data Sains

Dokumen resmi persetujuan proyek Optimasi Penjualan UMKM Keripik Nusantara menggunakan Data Sains.

LEMBAR PENGESAHAN

Proyek Data Sains: Optimalisasi Penjualan UMKM Keripik Nusantara
Versi 1.0

Dokumen ini merupakan bentuk persetujuan resmi untuk memulai dan melaksanakan proyek Data Sains yang bertujuan meningkatkan performa penjualan UMKM melalui analisis data, peramalan penjualan, segmentasi pelanggan, dan rekomendasi produk.

1. Informasi Proyek

  • Nama Proyek: Optimalisasi Penjualan UMKM Keripik Nusantara
  • Jenis Proyek: Data Sains
  • Bidang: Teknik Informatika
  • Tujuan: Memaksimalkan penjualan melalui prediksi penjualan & analisis perilaku konsumen

2. Ruang Lingkup Proyek

  • Business Understanding
  • Data Collection & Integration
  • Data Preparation
  • Feature Engineering
  • Modeling (Forecasting, Clustering, Recommendation)
  • Model Evaluation
  • Deployment API & Dashboard

3. Deliverables Proyek

  • Dataset terstruktur & terdokumentasi
  • Model prediksi penjualan
  • Model segmentasi pelanggan
  • Dashboard performa produk
  • API deployment model
  • Laporan akhir & presentasi
