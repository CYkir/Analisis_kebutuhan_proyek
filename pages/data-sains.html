Data Science untuk UMKM Keripik Nusantara

Bussiness Understanding

Tujuan Proyek Data Science: UMKM Keripik Nusantara ingin meningkatkan penjualan dengan memanfaatkan data penjualan online dan offline

Permasalahan Utama:

Tidak tahu produk mana yang paling laku per daerah

Sulit memprediksi permintaan tiap bulan

Tidak tahu jam terbaik untuk live streaming

Tidak memahami karakter pelanggan

Tidak mengetahui ROI dari program afiliasi

Goal Data Science:

Memprediksi penjualan 30 hari ke depan

Menentukan produk best-seller berdasarkan lokasi

Menentukan jam optimal live streaming

Melakukan segmentasi pelanggan untuk target promo

Mengukur performa afiliasi

Data Collection

Sumber Deskripsi
Data transaksi Tanggal transaksi, produk, qty, harga, metode bayar
Data afiliasi ID afiliator, komisi, total view
Data live streaming Jumlah penonton, jam live, durasi
Marketplace data Traffic produk, rating, review
Data stok Sisa stok per gudang

Data Preparation

Langkah Persiapan

  • Membersihkan missing value
  • Menghapus duplikasi transaksi
  • Konversi format tanggal ke datetime
  • Normalisasi kolom harga
  • Menyatukan data offline dan online

Contoh Preprocessing

  • Encoding kategori produk
  • Standard scaler pada jumlah penjualan
  • Time indexing untuk forecasting

Modeling

Forecasting Penjualan

ARIMA / SARIMA

Model statistik untuk memprediksi pola time-series penjualan.

LSTM

Neural network yang cocok untuk pola penjualan jangka panjang.

Segmentasi Pelanggan

K-Means

Mengelompokkan pelanggan berdasarkan perilaku pembelian.

DBSCAN

Segmentasi berbasis densitas untuk data noise & kompleks.

Prediksi Produk Best-Seller

Random Forest

Model ensemble untuk memprediksi produk paling laku.

XGBoost

Model boosting yang sangat akurat untuk prediksi penjualan.

Model Evaluation

Forecasting

RMSE

Mengukur error rata-rata dari prediksi terhadap nilai aktual.

MAPE

Mengukur persentase error dari hasil forecasting.

Classification

Accuracy

Persentase prediksi yang benar dalam klasifikasi.

F1-Score

Keseimbangan antara precision dan recall.

ROC Curve

Evaluasi performa model berdasarkan True Positive Rate.

Clustering

Silhouette Score

Mengukur seberapa baik klaster terbentuk.
